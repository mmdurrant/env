# env
A collection of scripts for Linux to make my life easier.

Includes
* tmux mouse stuff
* tmux window layouts
* vimrc
* bashrc to bring it all together
* default profiles for systems I work on
* misc crap

```bash
eval `curl https://mmdurrant.github.io/env/mdenv.sh`
```
OR
```bash
wget https://mmdurrant.github.io/env/mdenv.sh
source mdenv.sh
```
